# Asteroid Hunter
### Unity 3D Game

#### 3D space shooter game made in Unity3D with C#.
More info below.

### Features
- 3d game world
- Player space-ship
- One Mothership
- Laser-bolt weapon
- Enemies spawning
- Enemies follow mothersip or player
- Enemies attack their targets, and do retreat moves
- Enemies basic path-finding i.e avoids objects
- ...
